# Developer Analysis Project

## Project Overview
This project analyzes the Stack Overflow Developer Survey dataset to understand trends in programming languages, salaries, experience levels, education, and job satisfaction.

## Objectives
1. Identify the most popular programming languages used by developers.
2. Analyze the relationship between work experience and salary.
3. Compare salary distribution across education levels.
4. Examine the relationship between job satisfaction and salary.
5. Explore correlations among salary, experience, and job satisfaction.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- Advanced Visualizations

## Project Files
- final_main.py -> main Python project code
- charts/ -> saved output visualizations
- README.md -> project documentation

## Dataset
Dataset is not uploaded due to size limitations. It can be downloaded from the official Stack Overflow Developer Survey website.

## Key Insights
- Python, JavaScript, and SQL are among the most widely used programming languages.
- Work experience shows a positive relationship with salary.
- Salary varies across education levels, but skills and experience also matter.
- Job satisfaction is influenced by more than salary alone.
- Popular languages also show strong earning potential.
